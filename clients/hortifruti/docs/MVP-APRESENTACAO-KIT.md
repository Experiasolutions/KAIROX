## 3. TEMPLATE DE ROI — PARA OUTROS COMÉRCIOS

Substituir [VARIÁVEIS] para cada loja:

```
📊 DIAGNÓSTICO RÁPIDO — [NOME DO NEGÓCIO]

HOJE (sem sistema):
• Mensagens no WhatsApp por dia: ~[X]
• Tempo gasto respondendo: ~[Y] horas/dia
• Clientes que desistem por demora: ~[Z]/mês
• Valor médio por cliente: R$[V]
• 💸 Receita perdida estimada: R$[Z × V]/mês

COM O SISTEMA EXPERIA:
• Resposta automática 24h — 0 clientes perdidos por demora
• Tempo recuperado: [Y] horas/dia = [Y × 30] horas/mês
• Clientes recuperados: ~[80% de Z]/mês
• 💰 Receita recuperada: R$[0.8 × Z × V]/mês

INVESTIMENTO: R$0 (permuta)
RETORNO: R$[receita recuperada]/mês + [Y] horas livres/dia
```

### Exemplo Real Pré-Calculado — Hortifruti

```
📊 DIAGNÓSTICO — HORTIFRUTI ELAINE

HOJE:
• ~30 mensagens/dia no grupo de descontos
• ~2 horas/dia respondendo WhatsApp
• ~8 clientes desistem/mês por falta de resposta
• Ticket médio: R$35
• 💸 Perdendo: R$280/mês

COM SAFRA:
• Resposta automática 24h
• 60 horas/mês recuperadas
• ~6 clientes recuperados/mês
• 💰 R$210/mês recuperados
• + Grupo de ofertas organizado = mais vendas

INVESTIMENTO: Permuta (frutas/verduras)
ROI: Infinito (custo zero em dinheiro)
```
