# 📱 Guia do Douglas — Como usar a Safra

> E aí Douglas! Aqui tá o guia da Safra, a assistente que vai te ajudar na 2ª unidade.
> Manda mensagem normal, como se fosse pra uma funcionária.

---

## O que a Safra faz

✅ Atende clientes da sua unidade automaticamente
✅ Controla estoque (você fala, ela anota)
✅ Acompanha entregas (marca quando sair, quando entregar)
✅ Te manda resumo do dia

## Comandos principais

| Você manda              | O que acontece        |
| :---------------------- | :-------------------- |
| "quanto tem de tomate?" | Mostra o estoque      |
| "chegou 80kg laranja"   | Registra entrada      |
| "saiu 20kg cenoura"     | Dá baixa              |
| "acabou rúcula"         | Zera o item           |
| "estoque baixo"         | O que tá acabando     |
| "entregas hoje"         | Lista pendentes       |
| "saiu entrega Maria"    | Marca como despachada |
| "entregue"              | Marca como entregue   |
| "como foi hoje?"        | Resumo do dia         |

## Dicas

💡 Não precisa escrever certinho — a Safra entende "acabô banana" ou "qts tem d tomate"
📱 Funciona o dia todo, mesmo fora da loja
🔒 Clientes não veem seus dados de gestão
