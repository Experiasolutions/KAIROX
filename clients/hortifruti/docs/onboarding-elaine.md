# 📱 Guia da Elaine — Como usar a Safra

> Oi Elaine! Este é o guia de como usar a Safra, sua assistente digital.
> É bem simples — é só mandar mensagem como se estivesse falando comigo!

---

## O que a Safra faz por você

✅ Atende clientes automaticamente (cardápio, pedidos, promoções)
✅ Controla seu estoque (você fala, ela anota)
✅ Cria promoções pro grupo de WhatsApp
✅ Te dá o resumo do dia toda noite

## Como falar com a Safra

Manda mensagem normal, como se fosse pra um funcionário:

| Você manda                      | A Safra faz                   |
| :------------------------------ | :---------------------------- |
| "quanto tem de banana?"         | Mostra a quantidade           |
| "chegou 50kg tomate"            | Anota a entrada               |
| "vendeu 30kg batata"            | Dá baixa                      |
| "acabou alface"                 | Zera e avisa pra repor        |
| "estoque baixo"                 | Lista o que tá acabando       |
| "como foi hoje?"                | Resumo de vendas e entregas   |
| "cria oferta de abacaxi a 3,99" | Faz o post e dispara no grupo |
| "entregas hoje"                 | Lista as entregas pendentes   |

## Coisas importantes

🔒 **Seus dados são privados** — clientes nunca veem informações de estoque
📱 **Funciona 24h** — mesmo quando você não tá na loja
💬 **Se errar** — é só falar "cancelar" ou "desfazer"

## Precisa de ajuda?

Fale com o Gabriel (seu parceiro de tecnologia) — ele configura tudo pra você! 😊
