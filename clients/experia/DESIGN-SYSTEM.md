# Experia Design System — "Control Room Edition" v1.0

## 🎨 Paleta de Cores (Emerald / Amber)

A estética é inspirada no conceito de **Governança Digital Autônoma**, focada em trazer controle e clareza para o dono do negócio.

| Token | Valor Hex | Uso Sugerido |
|-------|-----------|--------------|
| `--bg-root` | `#050508` | Fundo principal da página |
| `--bg-surface` | `#0c0c14` | Cards e seções secundárias |
| `--accent-emerald` | `#10b981` | Sucesso, Confiança, "Operando OK" |
| `--accent-amber` | `#f59e0b` | Destaque, Alerta, "Ação Necessária" |
| `--text-primary` | `#f8fafc` | Títulos e texto principal |
| `--text-secondary`| `#94a3b8` | Subtítulos e descrições |
| `--border-glass` | `rgba(255, 255, 255, 0.08)` | Bordas de componentes transparentes |

## Typography

- **Main Headers (Hero/H1):** Inter (Bold/ExtraBold)
- **Secondary Headers (H2/H3):** Inter (SemiBold)
- **Monospaced (Metrics/Badges):** JetBrains Mono
- **Body:** Inter (Regular)

## Aesthetics

- **Glassmorphism:** Uso intenso de `backdrop-filter: blur(16px)` e fundos semi-transparentes.
- **Cyber-Noir:** Sombras suaves, glow em cores neon (emerald para OK, amber para foco).
- **Control Room:** Bordas finas, grid sutil no fundo, elementos que remetem a monitores industriais modernos.
