# Plan: PS2 Game Transfer and Optimization (E: to G:)

The goal is to move PS2 games from `E:\` to the OPL-prepared `G:\` drive, ensuring files larger than 4GB are split and the drive is defragmented for functional execution.

## Proposed Changes

### [Component] Game Transfer and Splitting
I will identify games on `E:\` and categorize them by size. 
- **Small Files (< 4GB)**: Kept on the old FAT32 drive.
- **Large Files (> 4GB)**: Formatted a new drive (`G:\`) to exFAT. All large ISOs (GOW2, Jak 3, Midnight Club) will be transferred entirely to `G:\DVD` with no splitting required. An updated OPL (Grimdoomer fork) will be provided on `G:\` to support the exFAT format natively.

### [Component] Optimization
- **Defragmentation**: After all transfers, I will run the Windows `defrag` utility on `G:\` to ensure OPL's functional execution.

## Verification Plan

### Automated Steps
1. **Size Verification**: Check that all transferred files on `G:\` do not exceed 4GB.
2. **Structure Verification**: Confirm `ul.cfg` is present if splitting was done.
3. **Defrag Status**: Check the output of `defrag G: /A` to confirm fragmentation level is acceptable.

### Manual Verification
- User to test the games on the PS2 console.
