## 7. PITCH FRAMEWORK (5 passos — Bilhon Chronicles)

```
PASSO 1 — O CAOS (Antes)
  "Me conta como funciona quando um cliente entra em contato."
  → O cliente descreve. Você conta as etapas.
  → "Você tem 11 pontos de falha."

PASSO 2 — A ANATOMIA (Números)
  "Cada 'tô esperando' é um loop infinito."
  → "Com a Experia: 75% menos. Em 6 semanas."

PASSO 3 — OS EXECUTORES
  Agente = "a mente que decide por você"
  Worker = "o que faz sem errar"
  Analyst = "quem pesquisa e gera relatório"
  Clone = "sua voz, seu jeito, em escala"
  NUNCA chamar de IA, robô, automação ou bot.

PASSO 4 — O HANDOFF
  "80% dos atrasos = passagem de bastão."
  "Com a Experia: notificação automática, rastreabilidade total."

PASSO 5 — ANTES vs DEPOIS (Fecha)
  Manual: 17-25 dias / Com Experia: 4-6 dias / 75% redução
  "Mesmas tarefas. Execução radicalmente diferente."
```
