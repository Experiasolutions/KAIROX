## FILOSOFIA CENTRAL — DARK COCKPIT

> "Dark Cockpit é a filosofia visual do AIOX: um sistema 100% dark-first
> onde cada token de cor, componente e animação foi desenhado para
> ambientes de alta performance."

**Referência:** cockpits de aviação
**Princípio:** zero distração, foco total na informação
**Aplicação:** 100% dark-first — não tem modo claro

**O que isso significa na prática:**
- Fundo preto/escuro como padrão absoluto
- Informação em destaque, não decoração
- Cada elemento justifica sua existência
- Alta densidade de informação sem poluição visual
- Design para quem trabalha muito — não para quem olha vitrine

```
60+    componentes UI documentados
```

### Tokens prioritários para definir

Antes de qualquer componente, definir:
```css
/* Identidade */
--color-primary: [verde Experia ou cor principal]
--color-accent: [cor de destaque]
--color-bg: [fundo dark]
--color-surface: [cards e painéis]
--color-text: [texto principal]
--color-muted: [texto secundário]

/* Tipografia */
--font-display: [fonte de títulos]
--font-body: [fonte de corpo]
--font-mono: [fonte de código/dados]

/* Espaçamento */
--space-unit: 8px [base do grid]

/* Motion */
--transition-fast: 150ms ease
--transition-base: 250ms ease
--transition-slow: 400ms ease
```
