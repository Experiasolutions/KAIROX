# PRD — Experia Landing Page v1.0 (Consolidado)

## 🎯 Objetivo
Transformar curiosidade em leads qualificados através de uma interface premium que comunica **Governança Digital Autônoma**.

## 📑 Estrutura da Página (10 Seções)

1.  **HERO:** Headline agressiva + Sub funcional + CTA (Quiz).
2.  **CAOS (Problema):** Visualizar os 55 pontos de falha (Visual do Bilhon Chronicles).
3.  **MÉTRICAS:** Tabela comparativa Manual (17-25 dias) vs Experia (4-6 dias).
4.  **OS 4 EXECUTORES:** Apresentação do Agente, Analyst, Worker e Clone.
5.  **TIMELINE (3 Semanas):** Mapeamento → Configuração → Governança.
6.  **ANTES vs DEPOIS:** Demonstração visual de clareza operacional.
7.  **CALCULADORA DE DESPERDÍCIO:** Input simples para mostrar dinheiro deixado na mesa.
8.  **PROVA SOCIAL:** Depoimentos (Placeholders focados em donos de clínica/negócio local).
9.  **PLANOS:** Local / Regional / Industrial.
10. **CTA FINAL:** Reforço da promessa e link para o Quiz.

## 🛠️ Tecnologias
- Framework: Vite + React + TS
- Estilo: Vanilla CSS (Focado em Tokens)
- Animações: Framer Motion / CSS Transitions
- Deploy: Vercel (target: `frontend-experia.vercel.app`)

## 🎨 Branding
- Paleta: Emerald / Amber
- Fontes: Inter / JetBrains Mono
- Aesthetic: Cyber-Noir / Control Room
